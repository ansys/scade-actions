Ansys SCADE Actions documentation#

Ansys SCADE Actions contains a collection of GitHub Actions that projects in the Ansys SCADE ecosystem can use.

For information on available actions and how to use them, see each of the following cards:

Query actions

Retrieve SCADE installation directories and Python environments.

Tests actions

Set up Python environments for SCADE and run the test suite for a SCADE Python library.
