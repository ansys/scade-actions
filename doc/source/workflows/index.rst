Ansys SCADE reusable workflows
##############################

Ansys SCADE Actions contains a collection of
`GitHub reusable workflows <https://docs.github.com/en/actions/reference/reusable-workflows-reference>`_
that projects in the Ansys SCADE ecosystem can use.

These workflows rely on `Ansys SCADE Actions <https://actions.scade.docs.pyansys.com/>`_
and `Ansys actions <https://actions.docs.ansys.com/>`_.
